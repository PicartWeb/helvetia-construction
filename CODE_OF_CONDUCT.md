# Code of Conduct

## Our Standard

Picart Web aims to keep this repository professional, respectful, and useful for learning and portfolio presentation.

Expected behavior:

- Be respectful and constructive.
- Stay on topic.
- Give clear feedback.
- Respect the license and attribution requirements.
- Avoid harassment, insults, spam, or disruptive behavior.

Unacceptable behavior:

- Harassment or discriminatory language
- Threats or personal attacks
- Publishing private information
- Rebranding or redistributing the project against the license
- Repeated off-topic or promotional comments

## Enforcement

Picart Web may remove comments, close issues, reject pull requests, or block users who do not follow this code of conduct.
