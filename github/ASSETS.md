# GitHub Preview Assets

This folder is reserved for future repository preview assets.

Suggested files:

- `preview.png` - main repository preview image
- `preview.gif` - animated preview or interaction showcase
- `hero.png` - homepage hero screenshot
- `services.png` - Leistungen page screenshot
- `projects.png` - Projekte page screenshot
- `mobile.png` - mobile layout screenshot

These files are optional and are not required for the website to run.
