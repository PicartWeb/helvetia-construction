# Roadmap

This repository is a completed portfolio demo. Future improvements are planned as polish tasks, not as a redesign.

## Planned

- Add final preview screenshots in `github/`.
- Add compressed social preview images.
- Add Lighthouse score notes after production deployment.
- Add optional structured data enhancements for local business SEO.
- Add more project case-study copy variants for portfolio presentation.

## Under Consideration

- Additional page-level Open Graph images.
- More detailed accessibility review notes.
- Optional print stylesheet for project pages.

## Not Planned

- Backend integration.
- CMS migration.
- React, Vue, or other framework rewrite.
- Template marketplace distribution.
- Commercial relicensing without written Picart Web approval.
