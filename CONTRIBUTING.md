# Contributing

Thank you for your interest in Helvetia Construction.

This repository is primarily a Picart Web portfolio demo, so public contributions are not expected in the same way as an open-source product. Suggestions and respectful issue reports are welcome.

## Before Contributing

Please note that this project uses the custom Picart Web Portfolio License. Personal learning forks are allowed, but commercial use, template redistribution, rebranding, client use, and removal of credits are not permitted.

## Suggested Contributions

Helpful contributions may include:

- Typo fixes
- Accessibility notes
- Browser compatibility observations
- Documentation improvements
- Deployment notes for static hosting

## Pull Request Guidelines

If submitting a pull request:

1. Keep changes focused and minimal.
2. Do not redesign pages unless requested.
3. Do not remove Picart Web credits or licensing text.
4. Test responsive behavior when changing HTML, CSS, or JavaScript.
5. Explain what changed and why.

## Local Development

This is a static website. You can open `index.html` directly or serve the folder with any static file server.

No package install or build step is required.
